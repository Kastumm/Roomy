This is an application designed for a better approach to managing meetings rooms in an office building.

//Administrator priveleges.
    - Create rooms, with a distinctive name and with a maximum amount of people that can fit.
    - Delete a room that is no longer needed.
    - Edit a room that needs an update.
    - User functionality is available for admin as well.

//User functionality
    - Appoint meeting for a certain room (Maximum amount of time per meeting is 2h)
    - End a meeting, it is no longer needed.
    - Check available time in the calendar.
